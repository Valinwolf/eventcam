### EventCam iOS
This is my first Swift app and has been quite a learning experience. It was heavily AI assisted due to me learning as I went. I originally tried to use the MCamera library to handle the camera part, however I could not get around a bug when re-opening the camera so I decided to ditch the library and create from scratch. 

This app was my response to my opinion on the pricing model of POV Cam. I decided to give people the ability to use this app as a service or as a self-hosted option. The code within this repository is licensed under the GNU GPL 3.0 or higher. 

I hope you enjoy the usage of EventCam!
